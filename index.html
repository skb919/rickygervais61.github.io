    사용법

    1. 최소 숫자를 설정합니다.
    2. 최대 숫자를 설정합니다.
    3. 도구 설정[숫자효과, 중복제외, 연속뽑기, 제외할 숫자 규칙]을 합니다.
    4. 랜덤 숫자 뽑기 버튼을 누릅니다.

     

    도구 설정 안내

    • 숫자 표시 효과 : 숫자가 증가하는 효과입니다. 연속뽑기에서는 지원되지 않습니다.

       

    • 중복 제외 뽑기 : 뽑은 숫자(기록 기준)는 제외하고 뽑습니다.

       

    • 연속 뽑기 : 설정한 횟수만큼 연속으로 뽑을 수 있습니다.

      연속으로 뽑는경우 숫자 표시 효과는 생략됩니다.

       

      만약 최소 숫자를 1, 최대 숫자를 45, 중복 제외 뽑기, 연속 뽑기 횟수를 6으로 설정하면 로또번호 뽑기가 가능하겠지만, 로또 전용 툴이 편할겁니다.

      중복 제외하는 경우, 다시 뽑으려면 한번 뽑을때 마다 기록 초기화가 필요합니다.

      (초기화가 없다면? 1회: 45개중 6개, 2회: 39개중 6개, 3회: 33개중 6개, 4회 27개중 6개 ...)

       

    • 제외할 숫자

      제외할 숫자 규칙을 입력할 수 있습니다.

      숫자 범위에 포함되지 않거나, [99 ~ 1] 처럼 범위가 잘못된 경우 규칙은 무시(제거)됩니다.

      설정을 바꾸면 중복 제외 남은 숫자 목록이 초기화 됩니다.

       

      1. 각 제외할 숫자(또는 범위)를 콤마(쉼표)[ , ]로 구분해주세요.

        (예: 1, 2, 3, 4)

         

      2. 숫자 범위는 물결표[ ~ ]로 입력해주세요.

        (예: 1 ~ 4, 15 ~ 20)

     

    기록 안내

    1. 총 숫자 개수 : 최소 ~ 최대 숫자의 총 개수 입니다.
    2. 중복 제외 남은 숫자 : 총 숫자 개수에서 뽑은숫자를 제외하고 남은 개수입니다.
    3. 기록 저장 : 텍스트(*.txt)파일로 저장할 수 있습니다.
    4. 기록 초기화 : 뽑은 기록을 초기화 할 수 있습니다. 기록을 초기화 할 때 뽑은 목록도 같이 초기화 되므로 중복 제외값이 초기화됩니다.

     

    기타 안내

    • 경품 추첨이 필요한 경우 더 공정하게 뽑을수있는 실시간 추첨기를 사용해보세요!

      결과를 일정시간이후. 모두가 같은 시각 이후에 볼 수 있어 실시간이 아닌, 커뮤니티등에서 추첨할 때 좋습니다.

      다만, 결과는 숫자(몇 번째 당첨자)로 표기되므로 미리 번호를 붙이는 작업이 필요합니다.

     

    번외

    • 0~100을 뽑으면 무작위처럼 보이는데, 0~10,000을 뽑으면 왜 1,000 이상의 숫자만 나오는것 같을까요?

      비밀은 착시현상입니다.

      0 ~ 999는 1,000개로 1,000 ~ 10,000의 9,001개 보다 약 9배 적습니다.

      원래 90%의 확률로 높은 자리수가 나오는데 우리가 착각하는겁니다.

    • 비슷한 숫자가 계속떠요! 정말 랜덤 맞나요?

      당연하게도 비슷한(또는 동일한)숫자가 연속해서 뜰 수 있습니다.

      이는 독립시행으로 매 실행시 범위내 각각의 숫자가 모두 동일한 확률으로 되어있습니다.

      0~999를 뽑을때 처음에 1이 나왔는데, "중복 제외 뽑기" 기능이 활성화된 경우가 아니라면 다음번에 1이 나올확률은 1/1000 = 0.1%로 동일합니다.

      1이 나올확률이 0.1%인데 99가 나올확률도 동일하게 0.1%, 123이 나올확률또한 0.1%로 어떤 숫자가 나와도 랜덤하게 나온결과가 맞습니다.
